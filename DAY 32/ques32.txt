Number of BST From Array --- 
Difficulty: Hard

Company Tags --- 
> NPCI

Question - You are given an integer array arr[] containing distinct elements.

Your task is to return an array where the ith element denotes the number of unique BSTs formed when arr[i] is chosen as the root.

Examples :

Input: arr[] = [2, 1, 3]
Output: [1, 2, 2]
Explanation: ex1explain.webp

Input: arr[] = [2, 1]
Ouput: [1, 1]

Constraints:
1 ≤ arr.size() ≤ 6
1 ≤ arr[i] ≤ 15

Expected Complexities --- 
>> Time Complexity: O(n log n)
>> Auxiliary Space: O(n)
